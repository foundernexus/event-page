# FounderNexus Event Page

Static GitHub Pages mockup for a FounderNexus event landing page.

Reference event: **Pitch Deck Teardown, live** with Isaac Kato of Two Ravens VC.

## Deploy

1. Push this repo to `main`.
2. In GitHub, open **Settings -> Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Choose branch `main` and folder `/ (root)`.
5. Save.

The page will publish at:

`https://foundernexus.github.io/event-page/`

## Main Files

- `index.html` - the page GitHub Pages serves.
- `styles.css` - responsive styles and FounderNexus visual tokens.
- `assets/` - wordmarks and speaker headshot from the supplied export.
- `Event Detail.dc.html`, `Event Page.dc.html`, `index-src.html` - original generated reference files.

## Before Sharing Publicly

- Replace `href="#"` on registration, membership, calendar, and invite links.
- Confirm the Isaac Kato bio and credibility notes are approved for external use.
- Confirm the event access model: members only, qualified guests, or open registration.
- Confirm the final date/time and Zoom routing.
